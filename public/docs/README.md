# PseudoGen v1.2.0

Generador de números pseudoaleatorios para el curso de Simulación.

## Características
- Generador Congruencial Lineal
- Generador Congruencial Multiplicativo
- Generador de Registro de Desplazamiento (Feedback Shift Register)
- Pruebas de Uniformidad y Aleatoriedad
- Visualización de Resultados en tiempo real
- Exportación a Excel

## Instalación
```bash
npm install
npm run dev
```

## Uso
Configura los parámetros iniciales (semilla, constantes a, c, m) y presiona "Generar".
